ADR-016 Modelos de Dominio Preparados para Evolución

Estado: Aceptado

Problema
Los modelos demasiado simples requieren cambios frecuentes conforme crecen las funcionalidades.

Decisión
Diseñar los modelos con capacidad de extensión controlada (metadata, identificadores de conversación, métricas).

Consecuencias positivas
•	Menos refactorizaciones. 
•	Compatibilidad con nuevos proveedores. 
•	Preparado para memoria, RAG y agentes colaborativos.
