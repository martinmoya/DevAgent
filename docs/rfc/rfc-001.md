RFC-001 Prompt Engine

Estado: BORRADOR

Objetivo
Crear una forma estándar de enviar peticiones a cualquier LLM.

Motivación
Actualmente cada proveedor utiliza parámetros diferentes.

Ejemplo.
LM Studio
temperature
top_p
seed
max_tokens

OpenAI
reasoning
temperature
modalities
response_format

Ollama
keep_alive
stream
raw
options

Solución
Crear un objeto común.
PromptRequest
y otro
PromptResponse

Ventajas
Todos los agentes hablarán exactamente el mismo idioma.

Riesgos
Será necesario adaptar cada proveedor.

Estado
Aprobado
